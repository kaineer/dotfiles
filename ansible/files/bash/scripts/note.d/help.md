## Notetaking
```sh
 $ note new           # create and edit new note
 $ note clone         # get repository from github
 $ note sync          # commit, pull, push
 $ note bro[wse]      # open github with repository
 $ note x [...args]   # run ...args in local repository dir

### FILTERS
 $ note f filename_filter # find file by name
 $ note {d,kyo}       # open current day notes
 $ note m[mon[th]]    # open current month notes
 $ note y[ear]        # open current year notes
 $ note               # list all note files
```
